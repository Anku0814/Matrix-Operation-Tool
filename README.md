# Matrix Operations using NumPy

## Project Overview:
This project demonstrates fundamental *matrix operations in Python using NumPy*. It is designed as part of a Python internship task to showcase mathematical computing, array manipulation, and linear algebra concepts using a scientific library.

## Objectives:
- Understand matrix representation in Python  
- Perform core linear algebra operations  
- Work with NumPy arrays efficiently  
- Demonstrate mathematical computation using code

## Features:
The program performs the following matrix operations:
Matrix Addition  
Matrix Subtraction  
Matrix Multiplication (Dot Product)  
Transpose of a Matrix  
Determinant Calculation
